Saves the current color match to the project history. 

Creates a new project automatically if none exists. Validates reference and sample colors, calculates Î”E, generates tint recommendations, and adds entry to project history with user-\/provided notes and acceptance status.

After successful save\+:
\begin{DoxyItemize}
\item Match\+Notes are cleared to empty string
\item Is\+Match\+Accepted is reset to true
\item History entry is persisted to disk via repository
\end{DoxyItemize}

The Match Details section (UI) captures\+:
\begin{DoxyItemize}
\item Notes\+: User annotations (e.\+g., "{}\+Post-\/recalibration"{})
\item Is\+Accepted\+: Whether match met quality threshold Both are stored in Color\+History\+Entry for audit/reporting. 
\end{DoxyItemize}
