\doxysubsection{Detailed Description}
Represents color matching data and algorithms for visualization in LAB color space. 

This model serves as the data bridge between the View\+Model\textquotesingle{}s RGB color inputs and the perceptually-\/meaningful calculations needed for color matching guidance. It maintains reference (target) and sample (candidate) colors in LAB space and provides methods for\+:
\begin{DoxyItemize}
\item Calculating ΔE (color difference)
\item Generating tint recommendations (which direction to adjust)
\item Supporting real-\/time color matching feedback
\end{DoxyItemize}

{\bfseries{LAB Color Space Overview}}

LAB is a device-\/independent, perceptually uniform color space defined by the CIE\+:


\begin{DoxyItemize}
\item {\bfseries{L\texorpdfstring{$\ast$}{*} (Lightness)}}\+: 0-\/100, where 0 is black and 100 is white
\item {\bfseries{a\texorpdfstring{$\ast$}{*} (Red-\/\+Green)}}\+: -\/128 to +127, where negative = green, positive = red
\item {\bfseries{b\texorpdfstring{$\ast$}{*} (Yellow-\/\+Blue)}}\+: -\/128 to +127, where negative = blue, positive = yellow
\end{DoxyItemize}

{\bfseries{Perceptual Uniformity}}

A key advantage of LAB\+: Equal distance in LAB space ≈ Equal perceived color difference to human observers. This is NOT true for RGB (where (255,0,0) and (254,0,0) are imperceptible in difference, but (0,0,255) and (0,0,0) are obviously different).

{\bfseries{Color Difference Algorithm}}

ΔE = √((Δ\+L\texorpdfstring{$\ast$}{*})² + (Δa\texorpdfstring{$\ast$}{*})² + (Δb\texorpdfstring{$\ast$}{*})²)

This is the CIE76 formula (Euclidean distance in LAB space).

Interpretation\+:
\begin{DoxyItemize}
\item \texorpdfstring{$<$}{<} 1\+: Imperceptible to human eye
\item 1-\/2\+: Barely perceptible
\item 2-\/5\+: Perceptible but acceptable
\item 5-\/10\+: Significant, likely unacceptable
\item \texorpdfstring{$>$}{>} 10\+: Very obvious mismatch
\end{DoxyItemize}

Modern color science also uses CIE94 and CIEDE2000 (which weight channels differently), but CIE76 is adequate for this application and faster to compute.

{\bfseries{Tint Recommendation Algorithm}}

The recommendation system guides users on how to adjust sample colors to match reference\+:


\begin{DoxyEnumerate}
\item If \texorpdfstring{$\vert$}{|}Δ\+L\texorpdfstring{$\ast$}{*}\texorpdfstring{$\vert$}{|} + \texorpdfstring{$\vert$}{|}Δa\texorpdfstring{$\ast$}{*}\texorpdfstring{$\vert$}{|} + \texorpdfstring{$\vert$}{|}Δb\texorpdfstring{$\ast$}{*}\texorpdfstring{$\vert$}{|} \texorpdfstring{$<$}{<} 0.\+5\+: "{}\+Colors are very close"{} (imperceptible difference)
\item For each channel\+:
\begin{DoxyItemize}
\item a\texorpdfstring{$\ast$}{*} channel (Red-\/\+Green axis)\+:
\begin{DoxyItemize}
\item If Δa\texorpdfstring{$\ast$}{*} \texorpdfstring{$>$}{>} 5\+: Recommend "{}\+Add Red"{} (sample too green, needs red)
\item If Δa\texorpdfstring{$\ast$}{*} \texorpdfstring{$<$}{<} -\/5\+: Recommend "{}\+Add Green"{} (sample too red, needs green)
\end{DoxyItemize}
\item b\texorpdfstring{$\ast$}{*} channel (Yellow-\/\+Blue axis)\+:
\begin{DoxyItemize}
\item If Δb\texorpdfstring{$\ast$}{*} \texorpdfstring{$>$}{>} 5\+: Recommend "{}\+Add Yellow"{} (sample too blue, needs yellow)
\item If Δb\texorpdfstring{$\ast$}{*} \texorpdfstring{$<$}{<} -\/5\+: Recommend "{}\+Add Blue"{} (sample too yellow, needs blue)
\end{DoxyItemize}
\end{DoxyItemize}
\item Combine recommendations\+:
\begin{DoxyItemize}
\item Multi-\/axis\+: "{}\+Add Red, Add Yellow"{} (if both axes need adjustment \texorpdfstring{$>$}{>} 5)
\item Single-\/axis\+: "{}\+Add Red"{} (if only one axis needs adjustment \texorpdfstring{$>$}{>} 5)
\item Fine-\/tune\+: "{}\+Fine adjustments needed"{} (if changes needed but \texorpdfstring{$<$}{<} 5 on all axes)
\end{DoxyItemize}
\end{DoxyEnumerate}

Threshold Selection (5 units)\+: The threshold of 5 units was chosen empirically\+:
\begin{DoxyItemize}
\item Too low\+: Generates recommendations too frequently for imperceptible differences
\item Too high\+: Misses observable differences
\item 5 units\+: \texorpdfstring{$\sim$}{\string~}2 perceptible units on a\texorpdfstring{$\ast$}{*} or b\texorpdfstring{$\ast$}{*} axis, practical for paint/dye adjustment
\end{DoxyItemize}

{\bfseries{Recommendation Accuracy}}

Recommendations are designed to be intuitive for users without color science background\+:
\begin{DoxyItemize}
\item "{}\+Add Red"{}\+: Red pigment should be added to sample
\item "{}\+Add Green"{}\+: Green pigment (or reduce red) should be added to sample
\item "{}\+Add Yellow"{}\+: Yellow pigment should be added to sample
\item "{}\+Add Blue"{}\+: Blue pigment should be added to sample
\end{DoxyItemize}

This quadrant-\/based approach provides actionable guidance without requiring users to understand LAB space mathematics.

Generates a tint recommendation based on color differences in a\texorpdfstring{$\ast$}{*} and b\texorpdfstring{$\ast$}{*} channels. 

Analyzes the LAB color difference and provides intuitive guidance on how to adjust the sample color to match the reference. Recommendations are based on thresholds in the a\texorpdfstring{$\ast$}{*} (red-\/green) and b\texorpdfstring{$\ast$}{*} (yellow-\/blue) axes.

\begin{DoxyItemize}

\item "{}\+Add Green"{}\+: Sample is too red; add green (or reduce red intensity)
\item "{}\+Fine adjustments needed"{}\+: Small differences (\texorpdfstring{$<$}{<}5) on one or both axes
\item "{}\+Colors are very close"{}\+: Imperceptible difference (\texorpdfstring{$\vert$}{|}Δ\+L\texorpdfstring{$\ast$}{*}\texorpdfstring{$\vert$}{|} + \texorpdfstring{$\vert$}{|}Δa\texorpdfstring{$\ast$}{*}\texorpdfstring{$\vert$}{|} + \texorpdfstring{$\vert$}{|}Δb\texorpdfstring{$\ast$}{*}\texorpdfstring{$\vert$}{|} \texorpdfstring{$<$}{<} 0.\+5)
\item "{}\+Enter both colors"{}\+: Missing reference or sample color
\end{DoxyItemize}

\end{DoxyReturn}
