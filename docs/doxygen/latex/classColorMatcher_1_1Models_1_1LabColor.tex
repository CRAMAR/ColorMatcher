\doxysubsection{Detailed Description}
Represents a color in the CIE LAB (L\texorpdfstring{$\ast$}{*}a\texorpdfstring{$\ast$}{*}b\texorpdfstring{$\ast$}{*}) perceptual color space. 

LAB is a device-\/independent color space designed to be perceptually uniform, meaning equal distances in LAB space correspond to equal perceived color differences by human eyes. This makes it ideal for color matching applications where perceptual accuracy is critical.

CIE LAB color space characteristics\+:
\begin{DoxyItemize}
\item Device-\/independent color space (consistent across displays and media)
\item Perceptually uniform\+: equal steps in LAB correspond to equal perceptual differences
\item Three components\+: L\texorpdfstring{$\ast$}{*} (lightness), a\texorpdfstring{$\ast$}{*} (green-\/red), b\texorpdfstring{$\ast$}{*} (blue-\/yellow)
\item L\texorpdfstring{$\ast$}{*} ranges from 0 (black) to 100 (white)
\item a\texorpdfstring{$\ast$}{*} ranges from -\/128 (green) to 127 (red), neutral at 0
\item b\texorpdfstring{$\ast$}{*} ranges from -\/128 (blue) to 127 (yellow), neutral at 0
\item Standard illuminant\+: D65 (daylight), 2° observer angle
\end{DoxyItemize}

Color Difference (ΔE)\+: The Euclidean distance between two colors in LAB space represents the perceived color difference (ΔE).
\begin{DoxyItemize}
\item ΔE \texorpdfstring{$<$}{<} 1\+: Difference not perceptible by human eye
\item ΔE 1-\/2\+: Barely perceptible difference
\item ΔE 2-\/10\+: Noticeable difference
\item ΔE \texorpdfstring{$>$}{>} 10\+: Very obvious difference
\end{DoxyItemize}
