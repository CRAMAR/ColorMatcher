\doxysubsection{Detailed Description}
Base class for all view models in the \doxylink{namespaceColorMatcher}{Color\+Matcher} application. 

Inherits from MVVM Toolkit\textquotesingle{}s Observable\+Object, providing automatic INotify\+Property\+Changed support for property change notifications to the Avalonia UI framework.

{\bfseries{MVVM Pattern in \doxylink{namespaceColorMatcher}{Color\+Matcher}}}

This application uses the Model-\/\+View-\/\+View\+Model (MVVM) architectural pattern\+:


\begin{DoxyItemize}
\item {\bfseries{Model}}\+: Color data, business logic (Color\+Project, Color\+History\+Entry, Color\+Space\+Converter)
\item {\bfseries{View\+Model}}\+: State management, command handling (Main\+Window\+View\+Model)
\item {\bfseries{View}}\+: UI presentation layer (Avalonia XAML and code-\/behind)
\end{DoxyItemize}

{\bfseries{View\+Model\+Base Responsibilities}}

By inheriting from Observable\+Object, all view models automatically get\+:
\begin{DoxyEnumerate}
\item INotify\+Property\+Changed implementation for property change notifications
\item Relay command support via \mbox{[}Relay\+Command\mbox{]} attribute
\item Observable property support via \mbox{[}Observable\+Property\mbox{]} attribute
\item Built-\/in change detection for MVVM Toolkit features
\end{DoxyEnumerate}
