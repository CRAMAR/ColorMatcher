\doxysubsection{Detailed Description}
Represents a color matching project with reference and sample colors. 

A Color\+Project is the primary data container in \doxylink{namespaceColorMatcher}{Color\+Matcher}, holding all information related to a single color matching session including target (reference) color, evaluation (sample) colors, and complete history of all attempts.

Project Lifecycle\+:
\begin{DoxyEnumerate}
\item Created with unique ID, name, description
\item Reference color set (target paint/gelcoat color to match)
\item Sample colors added and compared (generates history entries)
\item Each comparison creates Color\+History\+Entry with delta-\/E and recommendation
\item Project persisted to JSON file for later retrieval
\item Can be updated with new samples or cleared to start over
\end{DoxyEnumerate}

Persistence\+:
\begin{DoxyItemize}
\item Projects are stored as JSON files with naming pattern\+: \{Project\+Id\}.json
\item File\+Color\+Repository handles all persistence operations
\item Modified\+At timestamp is automatically updated on save
\item Color\+History is preserved across sessions
\end{DoxyItemize}
