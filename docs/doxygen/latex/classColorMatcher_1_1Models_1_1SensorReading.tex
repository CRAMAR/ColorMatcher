\doxysubsection{Detailed Description}
Encapsulates a color measurement captured from a spectrophotometer or colorimeter sensor. 

Represents a single sensor reading with color values, quality metrics, and metadata. Acts as a data transfer object between sensor implementations and the application logic.

{\bfseries{Sensor\+Reading Lifecycle}}


\begin{DoxyEnumerate}
\item {\bfseries{Creation}}\+: Sensor implementation creates instance via Read\+Color\+Async()
\begin{DoxyItemize}
\item Passes measured Rgb\+Color (required)
\item Optionally sets Lab\+Color (calculated or from sensor)
\item Sets Timestamp (UTC time of measurement)
\end{DoxyItemize}
\item {\bfseries{Enrichment}}\+: Sensor adds metadata
\begin{DoxyItemize}
\item Device\+Id\+: Which physical sensor made the measurement
\item Quality\+Score\+: Confidence metric (0-\/100)
\item Metadata\+: Device-\/specific info (illumination angle, mode, etc)
\end{DoxyItemize}
\item {\bfseries{Validation}}\+: Application checks quality
\begin{DoxyItemize}
\item Examine Quality\+Score vs requirements
\item Decide to accept or retry measurement
\item Use Timestamp to order measurements in session
\end{DoxyItemize}
\item {\bfseries{Processing}}\+: Application converts to Color\+History\+Entry
\begin{DoxyItemize}
\item Extract RGB/\+LAB colors
\item Store with reference/sample colors
\item Preserve metadata in history notes
\end{DoxyItemize}
\end{DoxyEnumerate}

{\bfseries{Color Representation}}

Sensor\+Reading provides colors in two formats\+:


\begin{DoxyItemize}
\item {\bfseries{Rgb\+Color}}\+: Device-\/dependent additive color model
\begin{DoxyItemize}
\item Always present (measured directly from sensor)
\item Ranges 0-\/255 for each component
\item Format understood by UI and graphics systems
\end{DoxyItemize}
\item {\bfseries{Lab\+Color}}\+: Device-\/independent perceptually uniform space
\begin{DoxyItemize}
\item Optional (may be null if sensor doesn\textquotesingle{}t provide)
\item More useful for color difference calculations
\item If null, application can convert via Color\+Space\+Converter.\+Rgb\+To\+Lab()
\end{DoxyItemize}
\end{DoxyItemize}

{\bfseries{Quality Metrics}}

Quality\+Score (0-\/100) indicates measurement confidence\+:
\begin{DoxyItemize}
\item 100\+: Perfect, ideal conditions (rare)
\item 75-\/90\+: Good, acceptable for most applications
\item 50-\/75\+: Fair, some uncertainty
\item \texorpdfstring{$<$}{<}50\+: Poor, potentially unreliable
\end{DoxyItemize}

Typical quality variations\+:
\begin{DoxyItemize}
\item Surface quality\+: Smooth surfaces score higher than textured
\item Lighting\+: Stable lighting scores higher than variable
\item Calibration\+: Post-\/calibration measurements score higher
\item Angle\+: Measurements within 45° of normal score higher
\end{DoxyItemize}

{\bfseries{Metadata Field}}

Device-\/specific sensor metadata examples\+:
\begin{DoxyItemize}
\item "{}\+Illumination\+: D65, 2° Observer, 45° angle"{}
\item "{}\+Mode\+: Reflectance, Averaged 3 measurements"{}
\item "{}\+Calibration offset\+: +2.\+5\%"{}
\item "{}\+Reading \#42 in session"{} (for tracking order) 
\end{DoxyItemize}
