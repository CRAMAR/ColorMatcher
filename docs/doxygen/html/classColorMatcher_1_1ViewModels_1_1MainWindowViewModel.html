async Task ColorMatcher.ViewModels.MainWindowViewModel.SaveColorMatchAsync ( )
inline

Saves the current color match to the project history.

Creates a new project automatically if none exists. Validates reference and sample colors, calculates Î”E, generates tint recommendations, and adds entry to project history with user-provided notes and acceptance status.

After successful save:

  • MatchNotes are cleared to empty string
  • IsMatchAccepted is reset to true
  • History entry is persisted to disk via repository

The Match Details section (UI) captures:

  • Notes: User annotations (e.g., "Post-recalibration")
  • IsAccepted: Whether match met quality threshold Both are stored in ColorHistoryEntry for audit/reporting.
